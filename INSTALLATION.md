## خطوات التثبيت

### 1. Google Sheet
1. اذهب إلى https://sheets.google.com وأنشئ جدولاً جديداً
2. سمِّه: **نظام الوسائل — كلية الحقوق BBA**
3. انسخ الـ **ID** من الرابط

### 2. Google Cloud
1. اذهب إلى https://console.cloud.google.com
2. أنشئ مشروعاً جديداً
3. فعِّل: **Google Sheets API** + **Google Drive API**
4. IAM & Admin ← Service Accounts ← Create
5. أنشئ مفتاح **JSON** وحمِّله

### 3. منح الصلاحية
- انسخ `client_email` من ملف JSON
- في Google Sheet ← Share ← أضفه كـ **Editor**

### 4. ملء secrets.toml
```toml
spreadsheet_id = "ID من الخطوة 1"

[gcp_service_account]
# الصق محتوى ملف JSON هنا
type = "service_account"
project_id = "..."
private_key_id = "..."
private_key = "-----BEGIN RSA PRIVATE KEY-----\n...\n-----END RSA PRIVATE KEY-----\n"
client_email = "...@....iam.gserviceaccount.com"
client_id = "..."
auth_uri = "https://accounts.google.com/o/oauth2/auth"
token_uri = "https://oauth2.googleapis.com/token"
auth_provider_x509_cert_url = "https://www.googleapis.com/oauth2/v1/certs"
client_x509_cert_url = "..."
```

### 5. تشغيل محلي
```bash
pip install -r requirements.txt
streamlit run app.py
```

### 6. نشر Streamlit Cloud
1. ارفع على GitHub (بدون secrets.toml)
2. https://share.streamlit.io ← اختر app.py
3. Settings ← Secrets ← الصق محتوى secrets.toml

## دورة سند التحويل

```
رئيس المصلحة  ──►  مسؤول الوسائل  ──►  أمين المخزن  ──►  رئيس المصلحة
  [pending]          [approved]          [executed]         [received]
 يُنشئ الطلب      يوافق أو يرفض     ينفذ + يخصم        يستلم ويوقِّع
                                                               ↓
                                                    سند HTML للطباعة
```
